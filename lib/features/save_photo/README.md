## 1. Modul

save_photo — das Foto im gewählten Album speichern und bestätigen.

## 2. Zweck

Das Foto wird im gewählten Album gespeichert.

Der Nutzer bekommt eine klare Bestätigung: Das Foto ist gespeichert.

Dieses Modul kümmert sich nur darum.

## 3. Hauptaktion

„Zurück zur Kamera“

Es gibt nur diese eine Aktion.

## 4. Was sieht der Nutzer

- Oben: klarer Kontext
- Mitte: eine ruhige, gut lesbare Bestätigung („Gespeichert“)
- Unten: genau eine Hauptaktion

Alles ist einfach, ruhig und eindeutig.

## 5. Was passiert

Das Foto wird gespeichert.

Der Nutzer sieht eindeutig, dass es gespeichert wurde.

Mit der Hauptaktion kehrt der Nutzer zur Kamera zurück.

## 6. Gehört nicht hierhin

- Album auswählen oder wechseln
- neues Album anlegen
- Foto prüfen oder Entscheidung treffen
- Foto bearbeiten
- Filter oder Effekte
- technische Hinweise
- lange Texte
- mehrere gleichwertige Buttons

## 7. Offen

- Formulierung für den Titel prüfen
